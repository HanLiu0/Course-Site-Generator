Help

Our site generator will automate the process of creating our course Web site to greatly reduce the amount of time needed in planning a semester and update it as it progresses. In order to do that our application will provide a Web site template that all generated content will use where user entered data can be store in JSON files that can then be loaded using JavaScript.

Step 1: Add General Site Information

To start, enter the information about the class and instructor. Then choose what pages you want the web page to have.

Step 2: Add Syllabus Information

Enter data for the course syllabus. Note that the "topics", "outcomes", "textbooks", and "gradedComponents", section must be in Json format.

Step 3: Add Meeting Time Information

Add information about meeting times. The "+" sign means add a lecture/recitation/lab. Choose an item in the table and click "-" sign will remove the item.

Step 4: Add TAs and Office Hours

To start, enter the name and email addresses of all Teaching Assistants. Then Select a TA and then toggle time blocks to adjust reservations. You can choose the start time and end time of the office hours. Double click the TA to edit.

Step 5: Add Schedule Calender

Enter the type, date, title, topic, and link of the schedule item and add it. You can select an item to update it.

Step 6: Save the Site Data

Once you've completed your work, you can press the save button and give your file a name. Note that the name in the app text field will be the name used in the Recent Work list of the Welcome dialog.

Step 7: Export the Site

Once you've saved your work, you can export the site and see how it looks in a Web browser
